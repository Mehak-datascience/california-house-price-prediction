# California House Price Prediction

## Problem Statement
The goal of this project is to predict median house prices in California based on various housing features.

## Dataset
California Housing Dataset containing information such as location, population, median income, and house age.

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Steps Performed
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training (Linear Regression, Decision Tree, Random Forest)
- Model Evaluation

## Result
Random Forest model provided the best performance for house price prediction.
